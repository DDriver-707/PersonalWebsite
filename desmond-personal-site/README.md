# Desmond Driver Personal Website

Free GitHub Pages portfolio site for hiring managers.

## Publish with GitHub Pages
1. Create a new GitHub repo named `desmond-personal-site`.
2. Upload `index.html`, `styles.css`, and the `assets` folder.
3. Go to repo Settings > Pages.
4. Under Build and deployment, choose Deploy from a branch.
5. Select `main` and `/root`, then Save.
6. Your site link will look like `https://YOUR-USERNAME.github.io/desmond-personal-site/`.

## Edit
Update text in `index.html`. Update colors and spacing in `styles.css`.
